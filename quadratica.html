Função Quadrática

Definição Características Gráficos

Definição

A função quadrática, também chamada de função do segundo grau, é expressa como f(x) = ax² + bx + c ou y = ax² + bx + c, sendo que os coeficientes "a, b e c" números reais e "a" diferente de 0 (zero).

Propriedades

F (x) = Ax² + Bx + C

Onde:

  • a ≠ 0
  • a > 0 concavidade para cima
  • a < 0 concavidade para baixo
  • os coeficientes b e c podem ser iguais a zero
  • O c é a intercepção com o eixo y

Exemplos de Funções de segundo grau:

f (x) 2x² + 3x - 4

a = 2 b = 3 c = -4

f (x) x² + 5

a = 1 b = 0 c = 5

y = 3x² - x

a = 3 b = -1 c = 0

y = -2x² - 3x + 8

a = -2 b = -3 c = 8

Raízes

Pontos de intercepção no eixo x.

  • As raízes são calculadas a partir da fórmula de Bhaskára.
  • delta > 0 teremos 2 raízes diferentes.
  • delta = 0 as raízes serão iguais.
  • delta < 0 não terá intercepção com o eixo x.

Vértice

Condizem com o ponto mínimo e ponto máximo

O ponto vermelho representa o ponto mínimo. O ponto mínimo sempre será encontrado quando o gráfico tiver a concavidade para cima.

Em gráficos com a concavidade para baixo encontraremos o ponto maxímo. Que está representado no ponto branco

Yv = -delta/4a
