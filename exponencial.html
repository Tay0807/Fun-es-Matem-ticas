Função Exponencial

Definição Características Gráficos

Definição

Chama-se função exponencial qualquer função onde o expoente é uma incógnita, e a base seja diferente de 1 e maior que 0.

Características

Onde usamos?

Quanto ao seu uso, vemos muitos dados que são dados como exponencial, ou seja, cresce exponencialmente. Podemos ter como exemplos: o crescimento populcional; evolução de capital em juro composto; desenvolvimento de bactérias em uma colônia etc.

Propriedades

y = aˣ

  • a é a base
  • a > 0 e a ≠ 1
  • se a > 1, a função será crescente
  • se 0 < a < 1, a função será decrestente
  • A função encontas no eixo x

    • Exemplos de Funções exponenciais:

    f (x) = 4ˣ

    f (x) = (1/2)ˣ + 2

    y = 0,8ˣ

    Gráficos

    Crescente

    Uma função é dita como crescente quando "a" for maior que 1

    Decrescente

    Uma função é dita como decrescente quando 0 < "a" < 1
